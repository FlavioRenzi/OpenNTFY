# OpenNTFY

## Example usage

```bash
OpenNTFY "Test message"
```

```bash
sudo apt upgrade; OpenNTFY "Upgrade terminated on {N}"
```

```bash
python long_program | OpenNTFY "Program terminated with result:"
```

```bash
OpenNTFY -p 5m30s "watch ip address" "Ip at time {T} is:"
```
